# Nutricare

Proyecto Nutricare: app de nutrición personalizada para condiciones crónicas.

## Estructura del proyecto

- `frontend/`: aplicación React + Vite
- `backend/`: API Node.js + Express
- `docs/`: documentación del proyecto
- `scripts/`: scripts de apoyo

## Comandos

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm run dev
```

## Requisitos

- Node.js 18+ instalado
- npm disponible
